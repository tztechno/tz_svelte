
# install

---------------------------------------------
```
npx degit sveltejs/template my-svelte-app
cd my-svelte-app
npm install
npm run dev
```
---------------------------------------------

  Your application is ready~! 🚀

  - Local:      http://localhost:8080
  - Network:    Add `--host` to expose
---------------------------------------------

# original appを作る時

srcフォルダのApp.svelteを書き換える 再度ビルドを行う

```
cd my-svelte-app
npm run build
```
